# 💻 Cadastro de Ativos – Projeto PIM (UNIP)

Este projeto foi desenvolvido como parte do **Projeto Integrado Multidisciplinar (PIM)** do curso de **Análise e Desenvolvimento de Sistemas** da **Universidade Paulista – UNIP**.

Apesar de ser um projeto apresentado como parte de um trabalho em grupo, **todo o desenvolvimento foi feito de forma individual**, desde a idealização até a programação e testes finais.

---

## 📌 Objetivo do Sistema

Criar um **sistema de gestão de ativos** simples e funcional, utilizando a linguagem **Python** e a interface gráfica do **Tkinter**. O sistema simula a rotina de uma startup na gestão de seus equipamentos e recursos.

---

## ⚙️ Funcionalidades

O sistema permite ao usuário:

- ✅ Cadastrar ativos
- ✅ Listar todos os ativos cadastrados
- ✅ Consultar ativos por ID
- ✅ Editar informações dos ativos
- ✅ Alterar o status de ativos (Ativo/Inativo)
- ✅ Gerar relatórios básicos de ativos
- ✅ Encerrar o programa de forma segura

---

## 🗂️ Estrutura dos Cadastros

Cada ativo contém os seguintes campos obrigatórios:

- **ID**
- **Descrição**
- **Valor**
- **Quantidade**
- **Número da Nota Fiscal**
- **Status**: Ativo ou Inativo

---

## 🧪 Funcionalidades Detalhadas

### 1. Cadastrar Novo Ativo
O usuário informa os dados básicos e cadastra o ativo diretamente no sistema.

### 2. Listar Ativos
Abre uma janela com a lista de todos os ativos cadastrados e seus dados.

### 3. Consultar por ID
Busca um ativo específico com base no ID informado.

### 4. Alterar Status
Permite marcar uma ou mais unidades de um ativo como **ativo** ou **inativo**, conforme necessário.

### 5. Gerar Relatório
Cria um relatório individual com as informações completas de um ativo, por exemplo:

ID: #001
Descrição: AMD Athlon 3000G With Radeon Vega 3
Valor: R$3300.00
Quantidade: 10
Status: Ativo

Botão que fecha a aplicação de forma segura.
